# Student Management Dashboard v2

Ek simple aur responsive frontend dashboard jo HTML, CSS aur JavaScript se bana hai. Isme bina kisi backend ke students ke records manage kar sakte ho.

## ✨ Features
- **Add Student**: Naye students add karo
- **View Records**: Sabhi students ki list dekho  
- **Delete Records**: Zarurat na ho to record delete karo
- **Responsive Design**: Mobile aur desktop dono pe sahi chalta hai
- **No Setup Needed**: Direct browser me chalta hai

## 🛠️ Tech Stack
- **HTML5** - Page structure
- **CSS3** - Styling aur layout
- **JavaScript** - Logic aur functionality
